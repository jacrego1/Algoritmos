# Planificador de CPU (FCFS & Round-Robin) – Python OOP

Simulador de planificación de CPU desarrollado en Python aplicando Programación Orientada a Objetos (POO).  
Permite registrar procesos, planificarlos con los algoritmos **FCFS** y **Round-Robin**, y calcular métricas de rendimiento.  
Incluye persistencia en **JSON/CSV** y pruebas unitarias con **pytest**.

---

## Instalación
```bash
py -m venv .venv
.\.venv\Scripts\activate.bat
pip install -r requirements.txt

## Ejecución
python -m src.main

## Pruebas
python -m pytest -q
